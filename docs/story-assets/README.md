# Story Assets

This folder keeps visual material for future Love Strings Dashboard Reels, Shorts, posts, and behind-the-scenes content.

## Screenshot Workflow

Before starting development on a visible module/tab, capture:

- Desktop/browser screenshot.
- Mobile screenshot.

After finishing UI work, or backend work that visibly changes a UI element, capture the same two views again.

Save files under:

`docs/story-assets/screenshots/<module>/`

Use this naming pattern:

- `<yyyy-mm-dd>-<module>-before-desktop.png`
- `<yyyy-mm-dd>-<module>-before-mobile.png`
- `<yyyy-mm-dd>-<module>-after-desktop.png`
- `<yyyy-mm-dd>-<module>-after-mobile.png`

Then link the files from `docs/build-story-log.md` near the matching story beat or Reel idea.
