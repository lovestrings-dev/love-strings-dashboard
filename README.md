# Love Strings Dashboard

This folder is the working home for the Love Strings Sprint Dashboard project.

Start here:

- `PROJECT_BRIEF.md` - main product memory and concept
- `docs/` - product, architecture, decisions, and planning notes
- `imports/` - spreadsheets, platform exports, and screenshots used as inputs
- `data/` - raw and processed dashboard data
- `app/` - web application source code once implementation starts
- `scripts/` - import, cleanup, and automation scripts
- `notes/` - rough notes and brainstorming
